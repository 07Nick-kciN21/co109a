## mult.asm

## C程式碼

```
#include<stdio.h>

int main()
{
    int x=10;
    int y=4;
    int i=1;
                
                
    loop:	 		
        if(i>y)		goto EXIT;        
        x += x;		
        i += 1;		
        goto loop;		

        EXIT:
        printf("%d", x);
        return 0;	
}
```

## 機器語言

```
@10

@2

@4  //i=y
M=D

@4
D=M

D=M
@3

@4

(EXIT)
0;JMP
```
