```
CHIP And {
    IN a, b;
    OUT out;

    PARTS:
    Nand(a=a, b=b, out=ANandB);//NAND = AND + NOT
    Not(in=ANandB, out=out);// NAND + NOT = AND
    // Put your code here:
}
```
